how to add a project in git?
to check if directory is a got repo type :
git status
to initialize a repo :
git init

how to undo the changes in working directory?
git checkout -- git\ tutorial/file-renamed.txt
double dash is used to avoid confusion between branch and file to be overwritten.

how to undo the changes in staging area?
git reset HEAD unwanted\ file.txt

how to amend commit in repo?
This can be used on ammending data or commen of last commit. You can change only latest commit.
The command is :
git commit --amend -m " git commit amed example "


To undo changes, rather than amending, make new commits that undo changes.

How to removed untracked files?
git clean is the command
$ git clean
fatal: clean.requireForce defaults to true and neither -i, -n, nor -f given; refusing to clean

aksma@DESKTOP-0FV5PJQ MINGW64 ~/git/algorithms/Algorithms (master)
$ git clean -n
Would remove bad file.txt

git clean -n tells which files will be removed form current directory
git clean -d -n tells which files and dirs will be removed

git clean -d -f actually removes the directoris and files inside those directories.
git clean -f removes files which were specified in git clean -n

reverting a specific commit
git revert 588a57e1ccd92cdede363b3bdce42c7ab81dff88

git ignore

Note :  Please commit gitignore file in project root so that everyone using this project will be able to ignore the same kind of files
